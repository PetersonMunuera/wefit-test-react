## Instalando o projeto

Depois de clonar o repositório, instale as dependências:

```bash
npm install
# ou
yarn
```

Inicialize a fake API :

```bash
npm run server
# ou
yarn server
```

Em outro terminal execute o projeto:


```bash
npm run dev
# ou
yarn dev
```

Abra [http://localhost:3000](http://localhost:3000) no browser.
